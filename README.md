# Proyecto-POO
Proyecto de Programacion Orientada a Objetos, acerca de un sistema de matriculas de los alumnos de una titulación en una facultad.

## Funcionamiento
Este proyecto funciona como una aplicación de escritorio usando la interfaz gráfica de customtkinter, con un entorno virtual que funciona mediante el lenguaje de programación Python, especificamente el proyecto fue realizado con Python 3.13.7.

## ¿Qué hace la interfaz gráfica?
Consiste en un sistema de matriculas de un cierto instituto, en el cual puede ingresar los alumnos, profesores y administradores mediante su correo institucional. Cada ventana será distinta dependiendo el rol:

### Estudiantes

Se les mostrará una interfaz en la cual podrán ver sus cursos inscritos y le permitirá inscribir más cursos

### Profesores

Se les mostrará una interfaz en la cual podrán ver sus asignaturas las cuales realizan, permitiendo ver todos los alumnos inscritos a dicha asignatura y ver su información.

### Administradores

Son los que controlan la interfaz, pueden ver todos los alumnos, profesores y asignaturas. Además, pueden crear y eliminar alumnos, profesores y asignaturas. Tienen total control del sistema.
